# Dotfiles

Xorg & i3
```
sudo pacman -S xorg xorg-xinit xterm i3-wm ttf-dejavu i3status
```

Polybar, Rofi, picom, alacritty
```
sudo pacman -S polybar rofi picom alacritty
```

misc
```
sudo pacman -S light nitrogen
```

fonts
```
cp -rf fonts/ $HOME/.local/share/fonts
```
